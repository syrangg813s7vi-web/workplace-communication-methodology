## 🔍 核心洞见

本书的主要论点：

1. **沟通失败 = 认知模型不匹配**
   不是你表达不清楚，而是你用的逻辑与对方的认知框架不匹配

2. **情商 = 识别他人真实需求的能力**
   高情商的人能看清别人真正想要什么

3. **权力存在于所有改变事情的能力中**
   不仅是职位，还包括信息、资源、影响力、关系

4. **赋能 > 控制**
   长期有效的领导力来自赋能，而不是控制

5. **话语权 = 价值 + 信任**
   你说话有分量，不是因为职位，而是因为别人信任你
