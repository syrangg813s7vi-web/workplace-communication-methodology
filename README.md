# 职场沟通与人际关系方法论：从情商到领导力

## 📚 项目概述

这是一本从 **269 份真实职场材料** 中提炼出来的方法论书籍。

通过分析职场沟通、人际关系、危机应对、领导力等多个维度的案例，我们提炼出了一套**可在生活中直接应用的沟通体系**。

## 🎯 适合人群

- 希望提升职场竞争力的专业人士
- 想要改善人际关系的团队领导
- 有志于长期职业发展的工作者
- 对情商管理和自我成长感兴趣的人

## 📖 书籍结构

### 第一部分：世界观（3 章）
理解职场和人际关系的底层逻辑

### 第二部分：框架与方法论（13 章）
学习可操作的沟通体系

### 第三部分：案例库（1 章）
从 150+ 个真实案例中学习

### 第四部分：系统地图（1 章）
整合所有框架的个人操作系统 + 30 天行动计划
